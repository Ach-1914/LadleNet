# Explanation of the Pretrained DeepLabv3+ Model in LadleNet

The pretrained DeepLabv3+ model utilized in LadleNet+ is based on this project: https://github.com/VainF/DeepLabV3Plus-Pytorch. The DeepLabv3+ model in LadleNet+ is trained on the Cityscapes dataset, and it employs ResNet101 as its backbone network.
